# 基于矩阵分解的哈勃深空图像分析与重构

**摘要**：本项目旨在探索高级矩阵分解算法（主要包括奇异值分解 SVD、满秩分解、以及 Cholesky 三角分解）在超高分辨率计算机视觉任务中的应用。我们以哈勃太空望远镜拍摄的极高分辨率彩色星云图为研究对象，分别构建了“基于 SVD 的深空降噪模型”与“基于 ALS 的像素残缺修复（补全）模型”。本文档将详细阐述项目背后的数学原理与工程算法细节，以便团队成员快速理解核心思路并顺利参与后续开发。

---

## 模块一：基于 SVD 的图像降噪与重构

### 1. 核心问题背景
宇宙空间探测器传回的图像，往往伴随大量的光子噪声与宇宙射线辐射（通常呈现为高斯分布或椒盐噪声）。我们的任务是在尽量不破坏星系自然轮廓和星际尘埃细节的前提下，通过纯数学建模的方式滤除这些物理底噪。

在计算机中，一张单色（灰度）图像可以被视为一个由亮度像素值构成的 $m \times n$ 的实数矩阵 $A$。对于彩色星云图（RGB），则是由三个这样的二维矩阵合并构成的三维张量。我们首先从二维矩阵讲起。

### 2. 数学原理深度剖析：SVD (奇异值分解)

**SVD 的代数形式：**
线性代数学告诉我们，任意一个 $m \times n$ 的实矩阵 $A$ 都可以被分解为三个极具规律的矩阵的乘积：
$$A = U \Sigma V^T$$
- **$U$ (左奇异矩阵)**: 代表图像的“列基向量”（垂直方向特征空间）。
- **$V$ (右奇异矩阵)**: 其转置 $V^T$ 代表图像的“行基向量”（水平方向特征空间）。
- **$\Sigma$ (奇异值矩阵)**: 一个对角矩阵，对角线上的元素 $\sigma_i$ 被称为“奇异值”，默认按从大到小的顺序严格降序排列。

**奇异值的物理图像学意义（核心）：**
奇异值的大小，直接反映了对应基特征在原图像中包含的“信息能量”。
- **头部巨大的奇异值**：对应图像的“低频信息”（如星系的整体轮廓、大面积的平滑渐变光谱、图像的主干结构）。这部分区域的像素点高度线性相关，是构成图像的基石。
- **长尾微小的奇异值**：对应图像的“高频信息”。在这部分信息中，既包含了偶尔出现的星际尘埃尖锐边缘，也包含了**占绝大部分比例的随机、无规律的高斯噪声**。

**如何实现降噪降维？——截断 SVD (Truncated SVD)**
既然噪声的“能量”主要隐藏在微小、长尾的奇异值当中，我们可以采取“一刀切”的截断策略：人为地设定一个阈值（非零奇异值的保留个数）$k$，强制仅保留前 $k$ 个最大的奇异值，将剩余所有较小奇异值统统抹零。我们将得到的新奇异值矩阵设为 $\Sigma_k$。此时重构后的近似矩阵 $A_k$ 为：
$$A_k = U \Sigma_k V^T = \sum_{i=1}^k \sigma_i u_i v_i^T$$
根据 Eckart-Young-Mirsky 定理，这是在限定秩等于 $k$ 的条件下，对原矩阵能够做到的**最佳近似（最低弗罗贝尼乌斯范数误差）**。截断较小奇异值的过程，就在数学层面上起到了提取主成分、并**物理剥离散列高斯白噪**的作用。

### 3. 工程化实现与调优基准
* **通道拆解与堆叠**：标准的 SVD 无法直接用于 RGB 带来的 3D 张量 ($m \times n \times 3$)。我们在代码层面采用**分而治之**的策略，拆解为 R、G、B 独立子通道并行执行降噪，重组后再叠加并通过 `np.clip` 截断防止亮度溢出。
* **参数 $k$ 的艺术极值**：
  - 若 $k$ 取得太大：导致降维不够，噪声依旧被原样重构。
  - 若 $k$ 取得太小：由于过分丢失高频信息，图片边缘变得平滑模糊。
  - **评价体系制定**：团队综合考量**峰值信噪比(PSNR)**与**压缩率指标**。我们通过脚本遍历探测发现：把 $k$ 固定在 $150$ 左右时，我们达到了“人类视觉最优临界点 (Visual Sweet Spot)”，即在高效压制噪声的同时原汁原味保全了星云层次。

## 模块二：基于 ALS 与 Cholesky 分解的图像大面积修复

### 1. 破损图像填补问题定义
假设因太空异常辐射，接收到的信号导致画面发生极其严重的损坏，有高达 $60\%$ 的离散像素彻底丢失。我们将这种状态抽象成一个观测掩码矩阵 $W$（1 代表像素完好，0 代表像素空洞）。
如何依靠仅仅剩余 $40\%$ 的已知数据，自动“推理填补”出具有极深物理关联的原始星云图像？

### 2. 数学原理深度剖析：矩阵补全 (Matrix Completion)

**从低秩先验引出满秩分解：**
真实世界的高清图像绝非纯随机白噪音，相邻像素之间存在平缓的梯度、极高的自相关度，因此**自然图像整体可近似认为是一个低秩矩阵**。
我们将残缺图像矩阵记为 $R$。基于低秩假说，我们可以认为原图是由两个极其低维度的“瘦高特征信息”结合产生的。即：
$$R \approx U V^T$$
其中，未知的特征矩阵 $U \in \mathbb{R}^{m \times k}$，特征矩阵 $V \in \mathbb{R}^{n \times k}$。其中 $k$ 是我们选定好的模型秩。

**模型损失函数确立（目标方程）：**
我们只关注有观测值的地方拟合得准不准。为了防止参数过拟合成满秩、或者数值爆炸失控，我们强制引入了 $L_2$ 正则化，从而得到的最终目标损失函数为：
$$L(U, V) = \sum_{(i,j) \in W} (R_{i,j} - u_i v_j^T)^2 + \lambda (\|U\|_F^2 + \|V\|_F^2)$$
注：$\lambda$ 决定了惩罚强度，防止求解特征值时陷入死角。

### 3. 工程算法攻坚战：ALS (交替最小二乘法)

注意，上述方程内部有个二次相乘项 $u_i \cdot v_j^T$。因为 $U,V$ 都是未知数，相互乘积导致这实际上是个**非凸优化方程**，难以直接一脚油门求解全局极小值。
**ALS 的破局思路**在于使用“交替定海神针”，将不可解的非凸大问题，拆分成一系列标准的一元二次凸优化问题：
1. **随机初始化**：先给矩阵 $V$ 随手赋上一些平滑数据。
2. **固定并更新**：当 $V$ 是常量的情况下，带入目标函数。原本的两数相乘变成了非常简单的二次多项式线性拟合（岭回归），此时可以立刻求导，算出能使当前误差最小的最优 $U$。
3. **轮转**：拿到新鲜刚出炉的 $U$ 当死锁常量，反过头去用同样的手法求最新的 $V$。
4. **收敛**：交替重复步骤 2 和 3。损失会像楼梯一样逐步阶梯式降低，直到最后“平稳收敛”。

### 4. 极致的加速引擎：Cholesky 分解

这是我们在本项目代码架构中**最大的亮点**。当固定 $V$ 重新拟合计算 $U$ 的某个单一列分量 $u_i$ 时（为了达到极小点令导数=0），代数理论给出必须解出正规方程组：
$$(V_{obs}^T V_{obs} + \lambda I) u_i = V_{obs}^T r_{obs}$$
我们可以将左侧那一大坨括号矩阵简记为标量 $A$。此时面临解形如 $A x = b$ 的灾难性线性系统：
- **灾难的常规逆矩阵解法**：算 $A^{-1}$。面对哈勃深空动辄百万级别以上的超密集点阵，$O(N^3)$ 复杂度的矩阵求逆会立刻导致内存穿顶、或者受到严重的浮点数溢出与舍入误差污染。
- **神来之笔 Cholesky**：团队研究发现，由平方项+单位矩阵得到的结构 $A = V_{obs}^T V_{obs} + \lambda I$ 拥有一个极其高贵的数学性质——它是一个**严格的对称正定矩阵 (Symmetric Positive-Definite)**。
线性代数理论保证了这种矩阵必定能被无损、高效且安全地唯一拆解成一个下三角矩阵 $L$ 和它的转置的乘积：
$$A = L L^T$$
此时待求原系统彻底沦为 $L L^T x = b$。
- 第一步，解个简单的一元代入：$L y = b$ (前向替换计算)。
- 第二步，解个简单的一元代入：$L^T x = y$ (后向替换计算)。

在实际代码中，我们在高核循环体内引入了底层数值算子 `scipy.linalg.cho_factor` 与 `cho_solve`。这使得我们避开了所有矩阵逆运算的黑洞，在 $60\%$ 像素丢失的验证案例中，凭借几乎忽略不计的代数求解耗时，在不到 $10$ 次的大型外层 ALS 交替跌代后，MSE（均方误差）发生了断崖式的跳水奇迹，极其完好地在宏观视觉上重建了震撼的深空原貌。
